# web-1-1

이 폴더는 1학년 '웹개발입문' 수업에서 사용한 수업 정리와 실습 파일을 모아둔 저장소입니다. 개인 실습, 과제, 예제 코드와 수업 자료를 쉽게 찾아볼 수 있도록 정리해두었습니다.

디렉터리 구조
- images/: 수업·과제에 사용된 이미지
- test/: 실습용 테스트 파일 모음
- ticket/: 과제·티켓 관련 자료
- web01/ ~ web05/: 주차별(또는 강의별) 실습 폴더
- test_1217하헌휘.zip, test_1217하헌휘/: 개인 제출물 예시

파일명 규칙(권장)
- 과제 제출: 이름_학번_과제명.ext (예: 홍길동_20260001_과제1.zip)
- 강의 노트: week01_notes.md 또는 web01_readme.md

기여 방법
- 파일을 추가/수정한 뒤 커밋 메시지에 변경 내용을 간단히 적어주세요.
- 팀 또는 수업 제출 방식이 있다면 해당 규칙을 우선합니다.

라이선스
